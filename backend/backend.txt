1. Виртуальная среда


1.1. Создание виртуальной среды

$: python -m venv venv


1.2. Активация виртуальной среды

$: venv\Scripts\Activate.ps1


1.3. Деактивация виртуальной среды

$: deactivate


2. Django


2.1. Создание проекта Django

$: python django-admin startproject shop .


2.2. Создание главного приложения

$: python manage.py startapp mainapp


2.3. Применение миграций БД

$: python manage.py makemigrate
$: python manage.py migrate 

2.4. Создание суперпользователя

$: python manage.py createsuperuser
